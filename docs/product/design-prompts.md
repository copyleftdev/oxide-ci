# Design Prompts for Oxide CI

Use these prompts with AI design tools (Midjourney, DALL-E, Ideogram) or as briefs for designers.

---

## Logo Prompts

### Primary Logo
```
Minimalist logo for "Oxide CI", a developer-focused CI/CD platform.
Style: Modern tech, geometric, clean lines.
Colors: Orange (#E85D04), dark gray (#1A1A2E).
Concepts: Pipeline flow, transformation, speed, reliability.
Must work at 16x16 favicon size.
No gradients, flat design.
```

### Icon Variations
```
App icon for CI/CD build tool called "Oxide".
Concept 1: Stylized "O" with integrated pipeline/arrow motif.
Concept 2: Hexagonal shape with gear or circuit elements.
Concept 3: Flame or spark suggesting compilation/build process.
Style: Flat, geometric, tech-forward.
Primary color: Rust orange (#E85D04) on dark background.
```

### Wordmark
```
Wordmark logo for "Oxide CI".
Font style: Modern sans-serif, slightly technical feel.
The "O" should be distinctive - perhaps hollow or with inner element.
Colors: Dark gray text, orange accent on CI or period.
Clean, professional, developer-friendly.
```

## Microsite Hero Prompts

### Hero Illustration
```
Abstract illustration for CI/CD platform landing page.
Show: Code flowing through pipeline stages (build, test, deploy).
Style: Isometric or flat design, modern tech aesthetic.
Colors: Orange (#E85D04), blue (#3B82F6), dark gray (#1A1A2E).
Elements: Terminal windows, containers, cloud, checkmarks.
Mood: Fast, reliable, modern.
```

### Feature Graphics
```
Icon set for CI/CD platform features:
1. Pipeline - connected nodes showing workflow stages
2. Containers - stacked boxes or Docker-style whale
3. Security - shield with lock or key
4. Speed - lightning bolt or stopwatch
5. Nix - snowflake or reproducibility symbol
6. Cloud - multi-cloud icons (AWS, GCP, Azure)
Style: Outline icons, 2px stroke, rounded corners.
Color: Single color (orange) or two-tone.
```
