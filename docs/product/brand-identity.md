# Oxide CI Brand Identity

## Brand Essence

### Name: **Oxide**
- Evokes **transformation** (oxidation = chemical change)
- Suggests **durability** (metal oxide coatings protect)
- Implies **simplicity** (oxide = base compound)
- Rust connection (oxide layer, Rust language)

### Tagline Options
1. **"Build with confidence."**
2. **"CI that scales with you."**
3. **"From commit to production. Fast."**
4. **"Reproducible builds. Zero trust."**

### Brand Personality
- **Technical** — Built by developers, for developers
- **Reliable** — Rock-solid, predictable, boring (in a good way)
- **Modern** — Rust-native, cloud-native, spec-driven
- **Open** — Open source core, transparent pricing

### Logo Concepts
1. **Hexagon + Gear** — DevOps + efficiency
2. **Flame + Circuit** — Build + automation
3. **O with pipeline** — Oxide + CI flow
4. **Rust-colored swoosh** — Speed + transformation

## Voice & Tone

### Writing Principles
- **Clear over clever** — No jargon unless necessary
- **Concise** — Respect developer time
- **Confident** — We know CI/CD
- **Helpful** — Solve problems, don't sell

### Example Copy

❌ "Leverage our cutting-edge CI/CD platform to synergize your DevOps workflows"

✅ "Ship faster with builds that just work."

## Positioning Statement

> **For development teams** who need reliable, fast CI/CD,
> **Oxide CI** is a **self-hosted-first build platform**
> that provides **reproducible builds with VM-level isolation**.
> Unlike GitHub Actions or CircleCI,
> we offer **Nix-native environments** and **keyless cloud authentication**
> with **transparent, usage-based pricing**.
